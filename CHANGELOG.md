# Changelog

## v0.1.0
- Initial tagged release: Tasktui with list, board, calendar, and detail views.
- Tags, checklists, due dates, timers with presets, and per-task countdown state.
- Configurable themes/layout via `~/.tasktui/config.toml` (or `TASKTUI_HOME`).
- Corrupt JSON recovery with backup, default sample tasks, and timer debounce on saves.
- In-app help overlay (`?`), plus new CI, lint/type checks, and dev extras.
